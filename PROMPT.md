## Prompts for Junie

---
Lets build an application that allows the user to select a file or directory, and then helps the user easily rename them with appropriate names.
A Sample workflow would be:
1. A user opens the app
2. A user clicks a button to select a file
3. The file is previewed and any text is selectable.
4. The app should scan the selected file for large text or dates and suggest a relevant name for the file.
	1. Suggestions could be generated by AI agents, or ideally some internal service
	2. Sample names:
		1. "Dwelling_Insurance_20250501.pdf" - for a pdf file that has the words "Dwelling Insurance" prominently at the top of the pdf, and the first found string that is a date is May 1, 2025.
		2. white_cat_20241002.jpg - an image file of a white cat that includes meta data indicating the photo was taken October 2, 2024, or the file creation date is October 2, 2024.
		3. Therapy_Homework_20260410.pdf - a file that includes "Therapy Homework" large at the top of the file, and with no date information readily available, uses todays date as the date string.
5. The user is shown a confirmation modal to confirm the name change.
6. If a directory of files is selected when the user clicks a button to select a file, as in step 2 of the workflow:
	1. the user will see each file one-by-one to rename.
